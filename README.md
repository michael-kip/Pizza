# PIZZA HUB

## Author

[Michael Kiprotich](https://github.com/michael-kip)

## Description

This is a web application that enables user to order for pizza online and have an option of home delivery if there is a need.

## Live Page

https://michael-kip.github.io/ip_four/

## Installation / Setup instruction

- Open Terminal {Ctrl+Alt+T}

- git clone `https://michael-kip/ip_four_`
- cd week_four_ip
- code . or atom . depending on the text editor of your choise.

## Requirements

- Either a computer,phone or a tablet.
- An access to the internet

## Technologies Used

- HTML
- CSS
- BOOTSTRAP
- JavaScript
- JQuery

## Reference

- HTML5

## BDD
